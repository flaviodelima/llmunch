# llmunch
A tool for prepping software project data for LLMs

## Usage

```bash
llmunch -o output.txt -d /path/to/project -e py txt
```

## Features
- Recursive directory scanning
- Multiple extension support
- Clean file markers
- EOF

## Installation
```bash
pip install .
```

or
```bash
pip install -e .
```
for development (editable install)
